## Lesson Content

你确实在基础知识方面做得很好。我们才刚刚开始；既然你已经学会了爬行，在下一组课程中，我将教你如何行走。

现在，你可以拍拍自己的背，休息一下。要退出 shell，你可以使用 `exit` 命令：

```bash
exit
```

或者 `logout` 命令：

```bash
logout
```

或者如果你正在使用终端 GUI，你可以直接关闭终端。下节课再见！
