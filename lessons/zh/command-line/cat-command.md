## Lesson Content

我们几乎完成了文件导航，但首先，让我们学习如何读取文件。一个简单的命令是 `cat` 命令，它是 concatenate 的缩写。它不仅显示文件内容，还可以组合多个文件并显示它们的输出。

```bash
cat dogfile birdfile
```

它不适合查看大文件，只适用于短内容。我们还有许多其他工具可以查看更大的文本文件，我们将在下一课中讨论。
